• Go to slack and press Create New App
  • Use the following manifest.yaml
    • _metadata:
  major_version: 1
  minor_version: 1
display_information:
  name: Cyclic Starter Bot
features:
  bot_user:
    display_name: Cyclic Starter Bot
    always_online: false
oauth_config:
  scopes:
    bot:
      - chat:write
      - channels:history
      - groups:history
      - im:history
      - mpim:history
      - app_mentions:read
settings:
  event_subscriptions:
    request_url: https://some-subdomain.cyclic.app/events
    bot_events:
      - app_mention
      - message.channels
  org_deploy_enabled: false
  socket_mode_enabled: false
  token_rotation_enabled: false
  • Set each of these environment vars:
    • process.env.SLACK_APP_CREDENTIALS_SIGNING_SECRET=123....long...hex...string...DEF
    • process.env.SLACK_BOT_USER_OAUTH_TOKEN=xoxb-123.......890-123.......890-AbC....alphanumeric....0Z
